SimpleCDN System Folder

Static files for the SimpleCDN System are served from this directory.

No external applications should depend on these files, as they may change without announcement in any release.

This directory contains files like the style sheet and the SimpleCDN logo.

By the way, if you see this page you are probably a developer. Nice! The SimpleCDN project is open source under the MIT license, and available on GitHub (JonathanBout/SimpleCDN). Feel free to have a look, ask questions or create a pull request.
